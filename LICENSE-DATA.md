# Data license

The `LICENSE` file in this repository (GPL-3.0) covers the **pipeline code** only — the shell/Python scripts, Planetiler configs, and CI workflow. It does **not** cover the `.pmtiles` and `manifest.json` files this pipeline produces and publishes as GitHub Release assets. Those are built from third-party geospatial data, and each stays under its own upstream license regardless of what license this repository's code carries. There is no single "data license" for this project — reuse terms depend on which file you're using.

This file exists so there's a clear, standalone answer to "what can I do with the released data" ([issue #1](https://github.com/nynosy/adiresy-tiles/issues/1)) without having to reconstruct it from the more discursive build-rationale notes in the README.

## Base map, buildings overlay, POI overlay — ODbL

Files: `madagascar-*.pmtiles`, `province-*.pmtiles` (base map); `buildings-*.pmtiles`; `poi-*.pmtiles`.

Built from OpenStreetMap data via [Geofabrik](https://download.geofabrik.de/)'s Madagascar extract. The buildings overlay additionally merges in [Google Open Buildings v3](https://sites.research.google/gr/open-buildings/) and Microsoft [GlobalMLBuildingFootprints](https://github.com/microsoft/GlobalMLBuildingFootprints) via [VIDA](https://source.coop/vida/google-microsoft-osm-open-buildings) — also ODbL.

Licensed under the **Open Database License (ODbL) v1.0** — full text: https://opendatacommons.org/licenses/odbl/1-0/. In short:
- **You may** use, copy, distribute, transmit, and adapt this data for any purpose, including commercially.
- **You must attribute** — see the credit line below.
- **Share-alike:** if you publicly use a database that's a substantial, non-trivial adaptation of this data, you must offer *that* derivative under ODbL too. Simply displaying the map (e.g., in an app) does not require this; republishing a modified copy of the underlying data does.

Required credit (also covers the base map's OpenMapTiles schema, CC-BY — see below):
> © OpenMapTiles © OpenStreetMap contributors

## Administrative boundaries overlay — CC BY-IGO 3.0

File: `boundaries.pmtiles`.

Sourced from Madagascar's National Disaster Management Office (BNGRC), via OCHA Field Information Services Section (FISS)'s HDX [COD-AB Madagascar dataset](https://data.humdata.org/dataset/cod-ab-mdg). **A different license from everything else in this repo** — not ODbL.

Licensed under **Creative Commons Attribution for Intergovernmental Organisations (CC BY-IGO 3.0)** — full text: http://creativecommons.org/licenses/by/3.0/igo/. In short:
- **You may** use, copy, distribute, and adapt this data for any purpose, including commercially.
- **You must attribute** — see the credit line below.
- **No share-alike requirement** — unlike ODbL, you are not obligated to release derivatives under the same license.
- **No implied endorsement:** the license explicitly prohibits suggesting BNGRC/OCHA endorses your use.
- **Dispute resolution:** per the license's own terms, disputes are resolved under UNCITRAL arbitration rules rather than a national court, reflecting BNGRC/OCHA's status as (part of) an intergovernmental body.

## manifest.json

The manifest itself (file names, sizes, checksums, dataset dates) is a factual index generated by this repo's GPL-3.0-licensed code, not a creative or database work — treat it the same as the code license.

## If you're only using one file

- Just the base map, buildings, or POI overlay → ODbL applies, use the OpenMapTiles/OSM credit above.
- Just `boundaries.pmtiles` → CC BY-IGO 3.0 applies, use the BNGRC/OCHA credit above.
- Both → both credit lines, since they're independent files under independent licenses (this pipeline does not merge them into a single tileset).
